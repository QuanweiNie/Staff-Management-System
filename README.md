##  Staff-Management-System
轻量级职工管理系统，包括老板、经理、员工的信息注册，及增删改查、排序等操作。

### Environment 
vs2019

### 使用说明
用vs新建项目，将所有文件放在项目目录下，编译即可。
该项目来源于B站黑马视频课，代码大约800行，并加入自己的一些改进和注释，是C++的入门级项目，可以很好地锻炼C++面向对象编程的知识。
